# SafeSound

Aplicativo para gerenciamento de pacientes, lembretes e rotinas desenvolvido durante o curso técnico.

**Principais pontos**

- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **Backend:** PHP (arquivos em `backend/`) com endpoints REST simples e banco MySQL

## Começar

Pré-requisitos:

- Node.js (para o frontend)
- PHP + MySQL (para o backend)

Passos mínimos para executar localmente:

```bash
# instalar dependências
npm install

# executar frontend (porta 5173 por padrão)
npm run dev

# servir backend via Apache / XAMPP (colocar o projeto em htdocs)
```

## Estrutura importante

- `src/` — código React (components, pages)
- `backend/` — endpoints PHP (`patients/`, `reminders/`, `routines/`, `cities/`)
- `docs/` — documentação do projeto (API e frontend)

## Documentação

Veja a pasta `docs/` para referência de API e instruções de desenvolvimento.

Contribuições são bem-vindas — veja `CONTRIBUTING.md`.
