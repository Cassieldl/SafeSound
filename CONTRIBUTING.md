# Contributing

Obrigado por contribuir! Algumas diretrizes rápidas:

- Faça um fork ou crie uma branch para cada feature/bugfix.
- Siga o padrão de commits informativos (ex.: `feat: add reminder form`, `fix: patients list pagination`).
- Teste localmente: `npm run dev` para frontend; garanta que o backend PHP esteja rodando.
- Abra um Pull Request descrevendo o que foi alterado e por quê.

Se quiser ajuda para preparar um ambiente de desenvolvimento, peça que eu gere instruções passo a passo.
